## Features

- Calculates annual carbon footprint (kg COâ‚‚/year)
- Transportation inputs (car, bus, train, metro, flights)
- Household electricity usage
- Cooking fuel type
- Diet type
- Shopping habits and recycling behavior
- Results displayed on a separate page
- Personalized reduction tips based on highest emission source
- Nature-inspired and responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Netlify

## How It Works

1. User enters lifestyle and usage details
2. Inputs are captured and frozen on submission
3. Emissions are calculated using predefined emission factors
4. Results are displayed on a separate results page
5. Personalized tips are shown to help reduce emissions
