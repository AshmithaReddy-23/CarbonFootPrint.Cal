# Carbon Footprint Calculator

A web-based Carbon Footprint Calculator that estimates a user’s annual CO₂ emissions based on everyday lifestyle choices.  
The application provides clear results, personalized tips, and an eco-friendly user interface to promote environmental awareness.

## Features

- Calculates annual carbon footprint (kg CO₂/year)
- Transportation inputs (car, bus, train, metro, flights)
- Household electricity usage
- Cooking fuel type
- Diet type
- Shopping habits and recycling behavior
- Results displayed on a separate page
- Personalized reduction tips based on highest emission source
- Nature-inspired and responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Netlify

## How It Works

1. User enters lifestyle and usage details
2. Inputs are captured and frozen on submission
3. Emissions are calculated using predefined emission factors
4. Results are displayed on a separate results page
5. Personalized tips are shown to help reduce emissions

## Getting Started (Run Locally)

### Prerequisites
- Node.js (LTS version)

### Installation
npm install

### Run the App
npm run dev

Open in browser:
http://localhost:5173

## Build for Production

npm run build

This creates a dist/ folder for deployment.
